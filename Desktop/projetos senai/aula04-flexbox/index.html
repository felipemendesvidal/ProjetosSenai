Fundamentos do flexbox

O css flexbox é um modelo de criação de layouts responsivos

Existe dois grupos dentro dessa criação, o Flex container e o Flex Item, que são os itens que estão dentro d container e como eles vão se comportar

Display flex

flex container

Display é uma propriedade antiga do css, com os atributos, inline, inline-block e agora Display:flex, apos isso, todos os elemntos dentro desse bloco se tornam flex item
